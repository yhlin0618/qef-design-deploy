# Bridge Review Log — `<source>.bridge.yaml`

> Audit trail for self-converging ensemble review (per `glue-bridge-self-converging-review` / #500).
> Each iteration of the codegen + review loop appends one `## Iteration N` section.
> Final outcome appears as `## Verdict: <CONVERGED|DIMINISHING|PLATEAUED|...>` on the last line.
>
> **Bound by**: MP102 v1.3 (Tier-2 Schema Contract — review provenance binds to bridge yaml).
> **Replaces**: legacy single-string `reviewed_by` in MP102 v1.2 (deprecated until 2026-07-31).
>
> Do not edit this file by hand. The `run_ensemble_review.R` helper (sibling to `validate_bridge_yaml.R`)
> appends iteration entries via `append_iteration_log()`. Manual edits will be overwritten on the next iteration.
